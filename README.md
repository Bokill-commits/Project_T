# 🚗 Project_T | 탁송대리기사 AI매칭시스템

AI와 데이터 기반으로 **탁송 차량과 대리기사를 최적으로 매칭**하고
**운행 경로 추천, 리뷰 감정 분석, 차량 손상 탐지, 기사 신뢰도 평가**를 수행하는 자동화 시스템 입니다.

## 📌 프로젝트 개요

기존 탁송 매칭 과정은 수작업 중심으로 이루어져 다음과 같은 문제가 존재합니다

* ⏱️ 배차 지연
* 📉 비효율적인 경로
* 💸 수익 구조 불투명
* 😡 고객 및 기사 만족도 저하

이를 해결하기 위해 **AI 기반 자동 매칭 시스템을 개발**하는 프로젝트를 진행했습니다.

---

## 🧠 주요 기능과 담당 기능

| 기능         | 설명             |
| ------------ | ------------------ |
| 🚀 **AI 자동 매칭**  | 기사와 화물 간 최적 조합 추천  |
| 🗺️ **경로 추천**    | 최적 운행 시각 및 경로 지도 제공   |
| ⭐ **리뷰 감정 분석**   | 고객 리뷰 기반 신뢰도 점수 반영 |
| 🔍 차량 손상 탐지  | 이미지 기반 차량 파손 여부 판단 |
| 📊 기사 신뢰도 평가 | 운행 이력 기반 점수 산출     |
| 🌐 웹 서비스     | 통합 플랫폼 제공          |

저는 **AI 자동 매칭과 경로 추천 / 리뷰 감정 분석 기능 구현**을 담당하였습니다.

## 🤖 4. AI 모듈 상세

### 🚗 운행 경로 및 자동 매칭 시스템

* **입력**
  * 기사 위치
  * 차량 정보
  * 출발지 / 도착지

* **처리**
  * 거리 및 시간 기반 스코어링
  * 조건 필터링
  * 우선순위 정렬

* **출력**
  * 기사 추천 리스트
  * 매칭 결과
  * 경로 지도

##

### 💬 리뷰 감정 분석 시스템

* **입력**
  * 고객 리뷰 텍스트

* **처리**
  * 형태소 분석
  * 텍스트 인코딩
  * AI 모델 학습 및 분류

* **출력**
  * 긍정 / 부정 분류
  * 신뢰도 점수 반영

##

### 🔍 차량 파손 탐지 시스템

* **입력**
  * 차량 이미지 (전/후)

* **처리**
  * 손상 영역 탐지
  * 파손 유형 분류

* **출력**
  * 파손 여부
  * 손상 데이터

##

### 📊 기사 신뢰도 평가 시스템

* **입력**
  * 운행 이력
  * 고객 평가
  * 파손 기록

* **처리**
  * 지표별 점수화
  * 가중치 적용

* **출력**
  * 종합 신뢰도 점수
  * 매칭 우선순위 반영
